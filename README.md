# Mídia Referência — Editor de Logotipo

Ferramenta web simples e responsiva para aplicar logotipo em lote sobre múltiplas fotos.

## Funcionalidades

- Upload de múltiplas fotos simultaneamente
- Upload do logotipo (PNG com transparência recomendado)
- Controle de tamanho do logotipo (5% a 100%)
- Controle de opacidade (10% a 100%)
- 9 posições rápidas (cantos, bordas e centro)
- Ajuste fino de posição por botões direcionais
- Arrastar o logotipo diretamente na prévia
- Exportar foto atual ou todas as fotos de uma vez
- Barra de progresso na exportação em lote
- Interface responsiva (desktop e mobile)

## Como usar

1. Acessar o site
2. Carregue as fotos desejadas
3. Carregue o logotipo
4. Ajuste tamanho, opacidade e posição no painel lateral
5. Clique em **Baixar TODAS as fotos** para exportar em lote
